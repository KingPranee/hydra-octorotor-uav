# Contributing

## Repo maintained by
Praneeth — SSN College of Engineering / HALO Aero Club

## How to contribute
1. Fork the repo
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Commit your changes with clear messages
4. Open a pull request — describe what changed and why

## Code style
- Python: follow PEP8, use type hints where practical
- C/C++: K&R brace style, 2-space indent
- Arduino: same as C++
- Commit messages: imperative present tense ("Add thrust calc", not "Added")

## Issues
Use GitHub Issues for bugs, questions, and feature requests.
Tag issues with: `bug`, `enhancement`, `documentation`, `hardware`
